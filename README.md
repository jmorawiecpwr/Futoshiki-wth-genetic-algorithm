A simple script solving standard futoshiki using a genetic algorithm. As part of a course at my university, we learned to solve similar problems using the CPlex solver, so I decided to implement the same problem metaheuristically in Python myself. This was also my first attempt at using this type of metaheuristic. My conclusions are as follows: on average, for small and medium-sized problems, heuristic algorithms, like the genetic algorithm, perform this type of task in a time similar to solvers like CPlex or Gurobi, but the exact time of a single invocation is much more difficult to predict. I consider this project a reference point for implementing such an algorithm; I hope it will be useful to someone.
